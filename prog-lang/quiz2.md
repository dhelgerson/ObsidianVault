Syntax: the form or structure of the code (expressions, statements, program units)
Semantics: the meaning of the code (what does each statement do?)
Alphabet: What characters can constitute a program (C++: letters, numbers,
symbols: ()[]{}.,;+=-*/, whitespace: space, tab, return, eof)
Sentence: string of characters from the alphabet
Language: set of sentences
Lexeme: the lowest level syntactic unit of a language (if, (, continue, ), etc.)
Token: category of lexeme: (keyword, punctuation, identifier, punctuation,
punctuation, identifier, operator, integer literal, punctuation, etc.)
